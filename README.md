# 仿网易云音乐桌面版
个人学习项目


#### 后台
[网易云音乐](https://github.com/Binaryify/NeteaseCloudMusicApi)

* 用户歌单单曲数，播放量可视化
* 音乐评论以弹幕的形式播放
* 以二维码形式分享音乐,扫码播放
* 控制，发送弹幕
* 播放列表自动播放
* 单曲，歌单，用户搜索
* 相似单曲推荐
* 点击播放条的音乐图片 展示歌曲详情
* 搜索历史
* 私人FM
* 获取私信,评论,通知
